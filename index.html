CSS Selectors Cheat Sheet

Simple Selectors

Selector Example Description Notes
ID #name Selects the element with id="name" Can only be used once per page. Highly specific, overriding less specific selectors.
class .name Selects all elements with class="name" Can be used multiple times in a page.
element h1 Selects the <h1> element Can be used to select any HTML element.
grouping .name, h1 Selects all elements with class="name" and the <h1> element Used to apply same styling rules to multiple selectors, making code more efficient and easier to read.
universal * Selects all HTML elements Commonly used to remove browser defaults. Low specificity.
